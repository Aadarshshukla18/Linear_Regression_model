# Linear_Regression_model
# 📊 Advertising Sales Prediction using Linear Regression

This project uses **Multiple Linear Regression** to predict product sales based on advertising budgets across different media channels.

---

## 🚀 Project Overview

The goal of this project is to analyze how advertising spending affects sales and build a machine learning model to predict future sales.

---

## 📁 Dataset Features

- TV Advertising Budget
- Radio Advertising Budget
- Newspaper Advertising Budget

### 🎯 Target Variable
- Sales

---

## 🧠 Model Used

- Linear Regression (from Scikit-learn)

---

## ⚙️ Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

---

## 🔍 Steps Performed

1. Loaded and cleaned the dataset  
2. Split data into training and testing sets  
3. Trained a Multiple Linear Regression model  
4. Evaluated performance using R² score  
5. Visualized relationships between features and sales  
6. Analyzed feature importance using model coefficients  

---

## 📈 Results

- The model achieved an R² score of around **0.48**
- TV advertising shows the strongest impact on sales
- Radio has moderate impact
- Newspaper has the least impact

---

## 📊 Visualizations

- TV vs Sales  
- Radio vs Sales  
- Newspaper vs Sales  

---

## 🧾 Conclusion

This project demonstrates how advertising channels influence sales. Among all features, **TV advertising contributes the most**, making it the most effective channel for increasing sales.
